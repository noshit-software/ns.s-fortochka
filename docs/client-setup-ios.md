# iOS Setup — v2RayTun

## 1. Install v2RayTun

Open the App Store on your iPhone and search for **v2RayTun**. Install it.

If you can't find it in the App Store, try **Hiddify** as a backup.

**Do this NOW** even if the VPN server isn't ready yet. If Apple removes the app later, you won't be able to install it.

## 2. Add the server (QR code method)

1. Open **v2RayTun**
2. Tap the **+** button
3. Choose **Scan QR Code**
4. Scan the QR code you received

## 3. Add the server (link method)

If you received a text link instead of a QR code:

1. **Copy the link** (it starts with `vless://`)
2. Open **v2RayTun**
3. Tap the **+** button
4. Choose **Import from clipboard**

## 4. Add subscription URL (recommended)

A subscription URL means the app automatically updates when servers change. You only do this once.

1. Open **v2RayTun**
2. Go to **Settings** (gear icon)
3. Tap **Subscription**
4. Tap **+** to add a new subscription
5. Paste the subscription URL you received
6. Tap **Update**

## 5. Connect

1. In v2RayTun, tap the server name to select it
2. Tap the **Connect** button (or toggle switch)
3. Allow the VPN permission when iOS asks
4. The VPN icon appears in the status bar when connected

## 6. Verify it works

Visit any website that shows your IP address. It should show the VPN server's IP, not your home IP. If you can access sites that were blocked before, it's working.

## Troubleshooting

- **Can't connect?** Try a different server if you have multiple in the subscription.
- **App asks for VPN permission again?** This is normal after updates. Allow it.
- **Slow speed?** The first connection may take a moment. If consistently slow, the server may be throttled — contact the person who set up your VPN.
