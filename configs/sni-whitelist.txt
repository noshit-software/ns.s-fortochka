# SNI domains known to work from Russia for VLESS+Reality
# These are major sites Russia cannot afford to block entirely.
# The DPI sees your connection as legitimate HTTPS to one of these domains.
#
# Rules for a good SNI target:
#   - Must be a domain Russia cannot afford to block (banking, enterprise, govt infrastructure)
#   - Must support TLS 1.3
#   - Must NOT be behind Cloudflare, Fastly, or other mass CDNs (CDN IPs break Reality)
#   - Must have a stable IP (direct A record, not CNAME-heavy CDN redirect)
#   - Russian-owned domestic domains are best for mobile ISPs (CIDR whitelist + SNI model)
#
# IMPORTANT — 2025-2026 context:
#   Some mobile ISPs (Beeline, MTS, Megafon) now use combined IP+SNI blocking:
#   your server IP must also be on their CIDR whitelist, OR you must use a
#   Russian IP range as an intermediate relay node. The SNI alone is not always enough.
#   For wireguard/home broadband users, SNI is still the primary gate.
#
# Run scripts/scan-sni.sh from a Russian VPS to verify which ones still work.
# Sources: github.com/igareck/vpn-configs-for-russia, github.com/hxehex/russia-mobile-internet-whitelist
# Last updated: 2026-04-06
